# Simple-MacOS

## Functions: ##
- **Navbar** with hover effects (would appear the current app name)
- Personal **LinkedIn** and **GitHub** links at the right side of Navbar
- Automatically updated **Clock**
- **Dock** with hover animation
- **Calculator** 
  - able to do basic calculations
- **Launchpad\:** 
  - click on the dock icon or double click on empty space to quit
  - able to open calculator and notes apps through launchpad
  - use searchbar to find app
- **Notes\:** 
  - delete: click on delete icon, and then click on selected note on sidebar/double click on selected note on sidebar
  - add: click on add icon
  - save and load changes
  - display sequence: last edit time
  - preview of selected note on the right
